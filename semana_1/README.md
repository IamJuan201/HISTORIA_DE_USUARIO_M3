## Description

This project is a simple interactive system made with JavaScript, the program asks the user for their name and age using prompt; then it checks if the age is a valid number if it is not a number it shows an error message, if the age is 18 or more it shows a message for adults, if the age is less than 18 it shows a message for minors.
The messages are shown in the console and in an alert.


## Features

- Ask the user for name and age.

- Validate if the age is a number.

- Show different messages depending on the age.

- Use console and alert to display messages.
